Сайт работает в соответствии с ДЗ по API - "12 REST запросы с requests. Работа с API".

Выбрал данный проект, так как у него уже были доработанные шаблоны (templates) и кэширование параметров.

У сайта есть 3 страницы:
* главная страница (HH API)
* страница использования API (Использовать HH API)
* страница контактов (Контакты)

В шаблоне по работе с API добавлен контейнер, в котором будут отображаться результаты API.
Также в шаблоне по работе с API есть форма, в которой можно ввести 4 параметра - текст поиска, номер страницы, регион и тип поиска (выгрузка в html)

В основном доработка была связана с внедрением backend-a на flask, т.к. до этого весь процесс работал из консоли.

Реализован HTML helper (format_dict_to_html()), чтобы красиво отображать JSON результат на странице.
